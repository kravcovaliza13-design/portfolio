Структура сайту

Сайт складається з трьох основних сторінок:

index.html — головна сторінка з привітанням та загальною інформацією
about.html — інформація про мене, освіта, навички
contacts.html — контактна інформація та форма зворотного зв’язку

portfolio/
├── index.html
├── about.html
├── contacts.html
├── images/
├── video/
└── audio/

Використані технології оці:
HTML5
Семантичні теги (header, main, section, footer)
Форми HTML (form, input, select, textarea)
Таблиці (table, thead, tbody)
Мультимедіа (img, video)
Git & GitHub

Проєкт розроблявся поступово з використанням Conventional Commits, наприклад:

init: initialize project
feat: add navigation menu
feat: create about page
feat: add contact form
feat: add multimedia elements
refactor: improve semantic structure
